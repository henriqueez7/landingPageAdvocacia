# Pedro Henrique Advocacia

Landing page institucional desenvolvida para portfólio, com foco em apresentação profissional, identidade visual elegante e experiência responsiva em desktop e mobile.

## Sobre o Projeto

Este projeto foi criado como uma landing page de portfólio para demonstrar a construção de uma página institucional moderna na área jurídica.

O objetivo foi desenvolver uma interface com:

- navegação simples e objetiva
- layout responsivo
- visual sofisticado e profissional
- seções pensadas para apresentação, autoridade e conversão

## Tecnologias Utilizadas

- React
- Vite
- Tailwind CSS 4
- CSS Modules
- Framer Motion
- Lucide React

## Funcionalidades

- navbar responsiva com menu mobile
- hero section com destaque visual
- seção de apresentação institucional
- cards de áreas de atuação
- seção de diferenciais
- bloco de conteúdo/artigos
- estatísticas e CTA final
- footer com links rápidos e informações de contato
- animações suaves de entrada ao rolar a página

## Responsividade

A landing page foi ajustada para funcionar bem em:

- celular
- tablet
- desktop

O foco da responsividade foi melhorar a usabilidade, a legibilidade e a organização visual em telas menores.
